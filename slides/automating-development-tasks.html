Automating Development Tasks

  • Start an instance of your application with a node server
  • Launch a browser window to see your running application
  • Watch for changes and run a task based on those changes
  • Reload browser after running tasks to reflect new changes
  • Lint new JS changes on save
  • Run unit tests for new JS changes on save
  • Enforce coding standards with cyclomatic complexity checking and jsbeautifying for code formatting

- End goal is to have a group of developers code look like one person wrote it
